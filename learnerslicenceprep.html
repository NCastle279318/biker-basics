Learners Licence Preparation

Course Duration = 6 hrs
(2 hrs per day for 3 days)

This is a theory class which covers all that is needed to pass the Official Learners Test:

1. Rules of the Road
2. Road Signs
3. Road Markings and...
4. Controls of the Vehicle

This course prepares you to write the Code 1 motorcyclist's learners licence test at the nearest Driver's Licence Testing Centre. Should you intend to take the learners test for any other code of vehicle, e.g. code 2 for light motor vehicles or code 3 for trucks, then please advise us when you book your preparation class and we will cover the content for those vehicles as well at no additional cost.

The course ends with a mock test using similar language to that used in the actual test. The course prepares the learner to pass the Code 1 (motorcycle) learners licence which allows the rider to be trained in the operation of a motorcycle.
